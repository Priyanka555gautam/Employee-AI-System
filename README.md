# Employee AI System

An AI-powered Employee Management System built using the MERN Stack (MongoDB, Express.js, React.js, Node.js) with JWT Authentication and AI Integration.

## Features

- Employee CRUD Operations
- JWT Authentication
- Password Encryption using bcrypt
- Protected Routes
- Employee Search by Department
- AI-based Employee Recommendations
- MongoDB Atlas Integration
- Professional React Frontend
- Responsive UI Design

---

## Tech Stack

### Frontend
- React.js
- Axios
- React Router DOM
- CSS3

### Backend
- Node.js
- Express.js
- MongoDB Atlas
- Mongoose
- JWT Authentication
- bcryptjs

### AI Integration
- OpenRouter API
- OpenAI GPT Model
